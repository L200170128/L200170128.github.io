# pgd-ui-kit-react

Contains all new PDS based on React

## Prerequisites

* Node v16.16.0
* NPM v8.11.0
* React-Native v18.2.0

## Installation

1. Clone the project

```bash
git clone https://repo.pegadaian.co.id/client/pgd-ui-kit.git
```

2. NPM Install

```bash
cd pgd-ui-kit
npm install
```

3. NPM Start

Ensure that the repository are pgd-ui-kit

```bash
npm start
```
